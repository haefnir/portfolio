JavaScript Game

22.04.2022

This project was completed as part of the EyUp coding academy during a 3-day sprint.
As our first group project we were tasked with producing a JavaScript game, loosely based on Minesweeper. We themed our game around removing gophers from a garden patch

I was responsible for generating a dynamic game board from the user inputs on the splash screen, with randomised locations for the gophers. I also handled win/loss logic, and developed the hinting system.
